Concepts

There are 3 elements in the program:

  • groups
  • items
  • details
The hierarchy is simple: a detail is a label with a text. The details must be inside an item. And the item must be inside a group. You can have as many groups as you like, and can put a group inside another group.

The groups are only for categorization. You don't need to use them (there is always a root group).

Groups and items are always showed in the left panel (or top panel, if in portrait mode), and the details always in right panel (or bottom).

You can resize the panels sliding the separator bar. The position of the bar is saved in a configuration file.

Buttons

The buttons are divided between the two panels. The 3 left buttons make operations in the groups and items:

  • : add a group inside the selected group
  • : add an item inside the selected group
  • : remove the selected item (and all its details) or the selected group (and all its items e details)

The buttons of the right panel have these functions:

  • : make a search
  • : move the selected detail up
  • : move the selected detail down
  • : add a detail
  • : show the detail labels in a list to easily remove or reorder it
  • : (only visible after pressed the edit button) remove the select details
  • : (only visible after pressed the edit button) close de detail labels

Editing

To edit the details you can enter the data in the field, or if the data are too large (or multi-line), double tap the text field and a larger field will be shown. To close press the back button ().

You can copy, cut and paste details, items and groups into another items and groups, and can copy the detail text into the clipboard to paste in another application. This is done via context menus. To show the menu make a long press in the detail, item or group.

To rename a group or item double tap it. To rename a detail show the detail labels in a list and double tap the label.
